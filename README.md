### 环境配置

正常安装 [DrRacket](https://racket-lang.org)后，某些代码用到了 `#lang sicp`，需要额外安装 sicp package。

打开 DrRacket, 选择 File -> Package Manager 菜单项。在 `Do What I Mean` 一栏输入 `sicp`, 点击 Install。
